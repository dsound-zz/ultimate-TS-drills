# Reverse String

## Problem Description

Write a function that takes a string and returns it reversed.

## Input Shape

- **Parameter**: `str: string` - A string of any length

## Expected Output

- **Returns**: `string` - The input string reversed

## Examples

```typescript
reverseString('hello')      // 'olleh'
reverseString('world')      // 'dlrow'
reverseString('a')          // 'a'
reverseString('')           // ''
reverseString('123')        // '321'
reverseString('racecar')    // 'racecar'
```

## Notes

- Handle edge cases: empty strings, single characters
- The function should work with any string content (letters, numbers, special characters)
